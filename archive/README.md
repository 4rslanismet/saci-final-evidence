# Archive

This directory contains clearly separated historical material that is not part of the canonical SACI publication dataset.

- `saci_lab_old_version/`: legacy laboratory portal snapshot.
- `maintenance/`: retired one-off site patch notes, scripts, figures, templates, and page snapshots.

Nothing under `archive/` should be used to calculate or validate the current SACI result. The canonical release is `docs/data/final/`.
