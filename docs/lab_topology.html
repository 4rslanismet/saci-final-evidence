Mimari katmanlarının anlamı

Katman Açıklama
Endpoints / Sources Kontrollü değerlendirme ortamındaki varlıkları temsil eder: domain controller, Windows endpoint, Linux host ve firewall.
Wazuh central components Logların toplanması, kuralların çalıştırılması, alert üretimi ve OpenSearch tarafına indekslenmesini gösterir.
SACI evaluation layer SACI engine deterministik skor hesaplar; evidence graph observed/missing ilişkileri üretir; policy-guided explanation katmanı ise bu çıktıları doğal dille açıklar.
Outputs Interactive evidence graph, explanation report, paper-clean görseller ve GitHub Pages üzerinden yayınlanan kanıt ekranlarıdır.
