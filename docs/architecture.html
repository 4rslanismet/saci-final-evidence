Adım Adım Kurulum (Single-Node Wazuh)Step-by-Step Installation (Single-Node Wazuh)

SACI laboratuvarı için adanmış sunucuya (Dedicated Server) kurulan Wazuh mimarisi adımları:Wazuh architecture steps installed on the dedicated server for the SACI laboratory:

# 1. Wazuh Installation Assistant İndirme
curl -sO https://packages.wazuh.com/4.14/wazuh-install.sh
curl -sO https://packages.wazuh.com/4.14/config.yml

# 2. config.yml Düzenlemesi (Tek node üzerinden 192.168.1.100)
nodes:
  indexer:
    - name: node-1
      ip: "192.168.1.100"
  server:
    - name: wazuh-1
      ip: "192.168.1.100"
  dashboard:
    - name: dashboard
      ip: "192.168.1.100"

# 3. Sertifika Üretimi ve Kurulum
bash wazuh-install.sh --generate-config-files
bash wazuh-install.sh --wazuh-indexer node-1
bash wazuh-install.sh --start-cluster
bash wazuh-install.sh --wazuh-server wazuh-1
bash wazuh-install.sh --wazuh-dashboard dashboard

# 4. Parolaları Çekme
tar -axf wazuh-install-files.tar wazuh-install-files/wazuh-passwords.txt -O | grep -P "'admin'" -A 1
