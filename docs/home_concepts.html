Öneri 2 — Akademik abstract sayfası

SACI: graph-based SIEM visibility evidence

SACI is an explainable graph-based scoring model designed to evaluate whether expected visibility relationships are observed across SIEM telemetry, detection controls, CTI enrichment and MITRE ATT&CK mapping.

The final evidence package reports complete graph closure in the declared evaluation scope: 97 nodes, 171 edges, 171 observed relations and 0 missing relations. The score is interpreted as a visibility closure indicator, not as an absolute security guarantee.
