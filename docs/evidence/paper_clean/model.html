SACI Conceptual Model

SACI evaluates whether expected visibility relationships are observable across telemetry sources, detection controls, MITRE coverage and CTI evidence.

Visibility Inputs→SOC Evidence Layer→Graph Scoring Core→Explanation Output

Visibility Inputs

Assets, log sources and expected telemetry relationships.

SOC Evidence Layer

Wazuh rules, alerts and detection-control evidence.

Graph Scoring Core

Observed and missing relationships are converted into SACI components.

Explanation Output

Policy-guided explanation interprets deterministic SACI outputs.

Formula: SACI = 0.30×LC + 0.25×CAC + 0.20×MDC + 0.15×CTIC + 0.10×TF.
