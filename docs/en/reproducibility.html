REPRODUCIBILITY

Reproducibility

A Vagrant-based scaffold reconstructs the declared laboratory topology, verifies the canonical evidence package and optionally prepares service deployment.

Boundary: This is a reconstruction scaffold, not a byte-identical archive of the original run. Live telemetry reproduction still requires provider-compatible guest images, an isolated authorized lab and runtime secrets.
Vagrantfiletopology.jsonREADMEverify.py

Profiles

evidence

Canonical artifact verification only.

portable

All declared machines as Linux stubs.

native

Linux, Windows and FreeBSD guests.

Fast verification

SACI_LAB_PROFILE=evidence vagrant up --provider=virtualbox

The wsiem guest receives docs/data/final and runs lab/verify.py without changing the published artifacts.

Native topology

export SACI_DC01_BOX='your/windows-server-box'
export SACI_WS01_BOX='your/windows-client-box'
export SACI_FW01_PFSENSE_BOX='your/freebsd-or-pfsense-box'
SACI_LAB_PROFILE=native SACI_DEPLOY_SERVICES=0 vagrant up --provider=vmware_desktop

ESXi

export GOAD_VAGRANT_ESXIHOST='192.0.2.10'
export GOAD_VAGRANT_ESXIUSER='root'
export GOAD_VAGRANT_ESXIPASSWORD='...'
export GOAD_VAGRANT_ESXIDATASTORE='datastore1'
export GOAD_VAGRANT_ESXINETWORK='SOC_VLAN'
SACI_LAB_PROFILE=native vagrant up --provider=vmware_esxi

Credentials are supplied at runtime and are never stored in the repository.
