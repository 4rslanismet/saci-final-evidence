References and technology stack

Official documentation, standards and academic literature for every platform and component used in the SACI laboratory, scoring chain, graph model and publication portal.

The page explicitly separates components directly used in the canonical final chain, infrastructure-only components, supporting tools and methodological sources.

Direct final chain

GOAD assets, Wazuh, Sysmon, Windows/Linux telemetry, pfSense, MISP, ATT&CK and SACI scripts.

Infrastructure

Vagrant and VMware ESXi used to deploy and operate the isolated lab.

Supporting components

OpenCTI was used during early CTI setup but is not part of the canonical final score.

Publication stack

Python, Requests, Cytoscape.js, Mermaid and GitHub Pages.
